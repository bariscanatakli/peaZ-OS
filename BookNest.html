BookNest - Book Recommendation System

Description: BookNest is a full-stack web application designed to help users find book recommendations based on their preferences. It uses the MERN stack to provide a seamless user experience.

Features

  • User authentication and authorization
  • Personalized book recommendations
  • Search functionality
  • Book reviews and ratings
  • Responsive design

Technologies Used

  • MongoDB
  • Express.js
  • React
  • Node.js
  • Redux
  • Bootstrap
